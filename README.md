# Queueing Theory M/M/c/K and M/M/1/K

Desktop application for calculating queueing Theory

![plot](./app.png)

## Getting Started

- Download this project (by click code and click download as a zip)
- Extract the zip
- Open dist folder
- Install Queueing Calculator Setup 0.1.0.exe
